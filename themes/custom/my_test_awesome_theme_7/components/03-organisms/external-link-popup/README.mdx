## How to use it in a Drupal site:

**Template**: `themes/custom/[theme_name]/templates/block/bootstrap-external-link-popup.html.twig`

1. Enable the theme `[theme_name]` or a sub-theme based on it.
2. Enable the contributed
   modules `External Link Popup` and `Bootstrap External Link Popup`.
3. Configure External link popup settings (`/admin/config/content/external_link_popup`).
